![Streamlit](https://img.shields.io/badge/Streamlit-App-red)

![NLP](https://img.shields.io/badge/NLP-SentenceTransformers-orange)

A **Retrieval-Augmented Generation (RAG) chatbot** that allows users to upload a PDF and ask questions about its content.
The system retrieves the most relevant text from the uploaded document using **semantic search**.

## 🚀 Live Demo

👉 https://samplechatbot-d2lnvgx2xewcz3c5d47fhb.streamlit.app/

## ⚡ Features

* Upload any **PDF document**
* Extract text from PDF
* Convert text into **vector embeddings**
* Store vectors in **ChromaDB**
* Retrieve relevant content using **semantic similarity**
* Answer questions based on the uploaded document

## 🛠 Tech Stack

* Python
* Streamlit
* LangChain
* ChromaDB
* Sentence Transformers
* PyMuPDF

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run chat.py
```
